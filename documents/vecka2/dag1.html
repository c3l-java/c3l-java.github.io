till play. * Skapa en ny webbapplikation. * Öppna din terminal. * Gå till skrivbordet. * Skriv play new play-test. * Detta skapar ett Play-projekt som heter play-test. * För att kunna anropa Play från andra ställen än just Play-mappen, så kan man lägga Play i vår PATH. * Förbered din applikation för att öppnas med Eclipse. * Öppna ett terminalfönster och gå till din applikationsmapp (i mitt fall play-test). * Starta Play genom att skriva play. * Om du inte gjort någonting med PATH, som vi sade att man kan göra, så måste du här skriva hela sökvägen. * Skriv i din Play-prompt: eclipse * Förbered Eclipse * För att använda Play på bästa sätt, och få direkt respons från Play när den kompilerar dina filer, så bör du installera Eclipse-pluginen för Play. * Importera till Eclipse * Starta nu Eclipse (för Java EE). * Hoppas du har installerat pluginen för Play. * Välj Eclipse → Preferences och klicka för “Refresh workspace on startup” och “Refresh using native hooks or polling”. * I ditt workspace, välj Import → Existing project. * Välj location: Skrivbordet, och sedan bör ditt projekt synas förikryssat. * Dölj alla externa .jar-filer så det inte blir så trångt * Kör igång servern. * Skriv ~run. * Besök [localhost:9000](http://localhost:9000). * Ta bort test/default package och alla filer däri * Testing är bra men kommer inte gynna oss just nu * Ta bort alla views** ###
